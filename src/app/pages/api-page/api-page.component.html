API – Документация
API SStats.net предоставляет данные о футбольных матчах (статистика, коэффициенты, составы, события и т.д) в формате JSON, csv.

Основной адрес: https://api.sstats.net

Пример:
Список матчей сезона: https://api.sstats.net/games/list?LeagueId=41&Year=2023
Данные матча: https://api.sstats.net/games/1183255
Данные прогноза Glicko 2 и xG: https://api.sstats.net/games/glicko/1183255

Авторизация
Для использования API вам необходимо иметь api-ключ, который представляет собой строку из 16 символов.

Публичный API ключ: 75kwgw7361l0l1ir. Ограничение: 300 запросов в минуту на всех пользователей.

API-ключ (apikey) должен передаваться в каждом запросе одним из трёх способов:
  1. В HTTP заголовках в поле apikey.
  2. В параметре адреса: ?apikey=75kwgw7361l0l1ir
  3. Либо поле apikey должно присутсвовать в Cookie

Получить индивидуальный API ключ можно после регистрации, на странице профиля
